# HTML 本地 PDF 萬用編輯工具

這是純 HTML + CSS + JavaScript 單頁架構，PDF 在瀏覽器本地處理。

## 啟動

```bash
python3 -m http.server 8765
```

開啟：<http://127.0.0.1:8765>

## 功能

- PDF 轉 PNG（ZIP）
- 左右/上下切割
- 抽出指定頁面
- 依頁碼分段拆分（ZIP）
- 每 N 頁拆分（ZIP）
- 合併多份 PDF（支援加入清單後批次合併）
- 旋轉頁面
- 刪除頁面
- 壓縮 PDF
  - 平衡壓縮：保留互動表單
  - 極限壓縮：重建結構 + 扁平化表單（非圖片化）
- 文字修改（覆蓋新增文字）
  - 可上傳 `.ttf/.otf` 字型支援中文

## 預覽

每個功能都提供 PDF 預覽，至少顯示首頁縮圖與頁數資訊；合併工具提供檔案清單預覽與移除。

## 頁碼格式

- `all`
- `1,3-5,7-end`
- `1-3;4-6;7-end`

## 技術依賴

透過 CDN 載入：

- `pdf-lib`
- `@pdf-lib/fontkit`
- `pdf.js`
- `JSZip`

第一次載入需網路；若要完全離線可改成本地 `vendor` 檔案。

## 限制

- 極限壓縮仍是 PDF 結構層壓縮，不包含掃描圖片重採樣。
- 文字修改目前是「覆蓋新增」，非 OCR 後語意替換原文字。
